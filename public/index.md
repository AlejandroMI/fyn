# Fyn — Find Your Nest

Fyn is an independent, open-source property-search service for Spain built for people and AI agents. Describe the home you need in natural language and Fyn searches multiple public property portals through one normalized MCP contract.

## Capabilities

- Search for property to buy or rent across multiple Spanish locations and portals.
- Apply structured constraints such as price, rooms, property type, floor, and lifestyle tags.
- Return source-attributed listing links, photos, match explanations, and coverage diagnostics.
- Connect over Streamable HTTP MCP at `https://fynfyn.top/mcp`.

## Access

No authentication, API key, account, or payment is currently required. Requests are rate-limited. Fyn does not reserve, sell, value, or legally verify property; verify every result on its original portal.
