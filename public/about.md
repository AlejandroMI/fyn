# About Fyn

Fyn (Find Your Nest) is an independent, open-source experiment maintained by Alejandro Marco. It provides one normalized MCP interface for discovering property across multiple public Spanish real-estate portals.

Fyn preserves source links and reports coverage limitations. It does not sell, reserve, value, or legally verify property and is not affiliated with the portals it searches. Source code is available at [github.com/AlejandroMI/fyn](https://github.com/AlejandroMI/fyn); contact `support@fynfyn.top` for operational or privacy questions.
