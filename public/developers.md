# Fyn developer guide

Connect an MCP Streamable HTTP client to `https://fynfyn.top/mcp`. The endpoint is public, anonymous, and rate-limited; no account, API key, OAuth token, or payment is currently required.

Run `tools/list`, then call `search_properties` with `city` or `locations` and structured constraints. Read coverage diagnostics, preserve source links, and verify listing details on the original portal.

See the [complete agent documentation](https://fynfyn.top/llms-full.txt), [OpenAPI description](https://fynfyn.top/openapi.json), [auth guide](https://fynfyn.top/auth.md), and [MCP server card](https://fynfyn.top/.well-known/mcp/server-card.json).
